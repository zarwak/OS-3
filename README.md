PA-03: Custom Linux Shell
Overview
This project implements a feature-rich Linux shell as per assignment requirements.

Features (Milestones)
Base Shell Loop and Prompt
Built-in Commands (cd, exit, help, jobs, history, set)
Command History and !n
Readline Integration
I/O Redirection and Pipes
Command Chaining and Background Jobs
If-Then-Else-Fi Support
Shell Variables (VAR=value, $VAR)
Git Workflow and Releases
Build and Run
make
./bin/myshell
